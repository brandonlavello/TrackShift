# Contributing

Thanks for your interest. This is a **personal side project**, not a commercial product.

## How to help

- **Bug reports:** Open an issue with a sanitized FIT/GPX sample if possible (remove personal info), expected vs actual behavior, and browser/OS.
- **Pull requests:** Keep scope focused. Match existing code style. Describe what changed and why.

## Development

```bash
npm install
npm run dev
```

Node 20+ (see `.nvmrc`).

## What we won't merge without discussion

- Backend services or file uploads (client-only is a core principle)
- Heavy dependencies for marginal UX wins
- Breaking changes to export format without a migration path

No response time guarantees. Maintainers merge when it makes sense.
